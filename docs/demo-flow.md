# 🌐 MARTHOS MVP 체험형 데모 시나리오 (v0.1)

## 🧪 시나리오 힘널
1. 사용자 `/donate` 접속 → 설명 입력 후 제출
2. 시스템에서 점수(`0.87`) 발포 + 기여자 인정 메시지 표시
3. `/dashboard`에서 점수, NFT 상태(`Minted ✅`), 트랜지텔션 하시 확인
4. Polygon Mumbai 링크 클릭 → 트랜지텔션 확인
